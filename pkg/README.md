# osc-cli packages maker

This directory contain methods to create osc-cli packages

## Make All

just type `make`

this will create all packages, for now only AppImage is supported

## Linux Appimage

```
make osc-cli-x86_64.AppImage
```

## Windows binary

```
make osc-cli-x86_64.zip
```

## clean

```
make clean
```
